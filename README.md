# 餐厅评论洞察分析工具

一个基于 React 的智能餐厅评论分析工具，帮助消费者快速了解餐厅口碑，帮助商家发现改进机会。

---

## 功能特性

### 消费者视角（默认）
- **餐厅口碑查询**：输入餐厅名称即可查看该餐厅的评论分析概要
- **热门餐厅推荐**：一键选择海底捞、外婆家、西贝、绿茶等热门餐厅
- **补充评价**：用户可以添加自己的评价，系统会实时重新分析
- **情感分布分析**：饼图展示正面/负面/中立评价比例
- **四维度分析**：菜品、服务、环境、价格四个维度的高频关键词和满意度
- **优缺点洞察**：Top3 优点和缺点一目了然
- **一键分享**：复制分析报告到剪贴板，方便分享给朋友

### 商家视角
- **数据仪表盘**：本月正面/负面评价、评论数、平均评分
- **趋势分析**：六个月四维度评分走势图
- **改进建议**：按优先级排序的具体改进方案
- **演示账号**：demo / 123456

## 示例分析报告

### 案例：海底捞火锅

**一段话总结**
> 【海底捞火锅】整体口碑正面。最受好评的是服务，主要槽点在于价格。优点：服务热情、服务周到、服务好；缺点：价格贵、价格偏高。适合对服务有要求的食客。

**情感分布**
- 正面评价：78%
- 负面评价：12%
- 中立评价：10%

**各维度满意度**
| 维度 | 满意度 |
|------|--------|
| 菜品 | 72% |
| 服务 | 85% |
| 环境 | 68% |
| 价格 | 45% |

**Top3 优点**
1. 服务热情（4次提及）
2. 服务周到（3次提及）
3. 等位服务好（2次提及）

**Top3 缺点**
1. 价格贵（5次提及）
2. 排队时间长（3次提及）
3. 人均偏高（2次提及）

**高频关键词**
- 菜品：番茄锅底、牛肉粒、虾滑、毛肚、新鲜
- 服务：热情、周到、帮忙、等位、美甲
- 环境：干净、整洁、舒适、宽敞
- 价格：人均150、贵、值得、性价比

### 核心算法实现

#### 1. 情感分析算法

基于情感词典的规则匹配方法：

```javascript
// 维度关键词词典结构
const DIMENSION_KEYWORDS = {
  菜品: {
    positive: ['好吃', '美味', '新鲜', '地道'...],
    negative: ['难吃', '咸', '淡', '腥'...],
    keywords: ['麻婆豆腐', '回锅肉', '口味'...]
  },
  // 服务、环境、价格同理
};

// 情感计算逻辑
1. 遍历四个维度
2. 检测是否提及该维度关键词
3. 统计正面/负面情感词出现次数
4. 比较正负词频确定情感倾向
5. 计算各维度正面比例
```

#### 2. 关键词提取算法

```javascript
// 基于词频统计
1. 合并所有评论文本
2. 对每个维度的关键词列表进行正则匹配
3. 统计每个关键词出现次数
4. 按词频降序排序，取 Top 5
```

#### 3. 优缺点提取算法

```javascript
// 结合情感词和维度词
1. 遍历每条评论
2. 检测维度关键词 + 正面/负面情感词组合
3. 记录 (维度 + 情感词) 作为优缺点
4. 统计出现次数，取 Top 3
```

### 数据管理

#### 预设餐厅数据
- 5 家热门餐厅预设真实模拟评论（海底捞、外婆家、西贝、绿茶、呷哺呷哺）
- 未预设餐厅自动生成随机评论数据

#### 历史记录
- 使用 localStorage 存储最近 10 条查询记录
- 支持快速加载历史分析结果

### 响应式设计

- 移动端优先设计
- 左右分栏布局，小屏幕自动堆叠
- 使用 Tailwind CSS 实现自适应
